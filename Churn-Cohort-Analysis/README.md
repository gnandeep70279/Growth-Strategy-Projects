# 📉 User Retention & Price Sensitivity Study

## Business Problem
High churn rates among new users were impacting long-term revenue. I needed to determine if specific user segments were more sensitive to pricing changes.

## Technical Approach
* **Cohort Analysis:** Segmented users by their "First Purchase Month."
* **Retention Modeling:** Built a pivot table in SQL to track the percentage of active users over a 12-month lifecycle.

## Insights & Impact
* **Discovery:** Users from Tier 2/3 cities showed a 40% higher churn rate when discounts expired.
* **Recommendation:** Implemented a "Dynamic Pricing" model offering small re-engagement discounts for churned users.
* **Projected Result:** Improved Month-3 retention by 15%.
