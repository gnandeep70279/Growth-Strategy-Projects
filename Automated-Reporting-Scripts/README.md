# ⚙️ Process Optimization: Automated KPI Pipeline

## Business Problem
The Marketing and Product teams were making frequent ad-hoc data requests, leading to manual errors and a 20% loss in analyst productivity.

## Technical Approach
* **Automation:** Developed a `Master View` in SQL that joins session logs and transaction data.
* **Data Integrity:** Implemented logic to filter out internal test accounts and failed transactions.

## Insights & Impact
* **Efficiency:** Reduced manual reporting time by 20% per week.
* **Accuracy:** Created a "Single Source of Truth" for company-wide KPIs (DAU, ARPU, and Conversion).
* **Impact:** Faster decision-making for cross-functional teams.
